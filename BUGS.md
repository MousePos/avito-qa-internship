Сценарий: Удалить номер с несуществующим номером, но соответствующий по формату

Ожидаемый результат:

```
{
    "result": {
        "message": "",
        "messages": null
    },
    "status": "404"
}
```

```
{
    "result": {
        "message": "",
        "messages": null
    },
    "status": "500"
}
```
