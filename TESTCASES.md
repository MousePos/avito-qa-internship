# ТЕСТ-КЕЙСЫ

1. Создать объявление
Желаемый результат: 200	Success response

2. Получить объявление по его идентификатору
Желаемый результат: JSON объявления с указанным id

3. Получить все объявления по идентификатору продавца
Желаемый результат: Все объявления, созданные указанным SellerID

4. Получить статистику по id объявления
Желаемый результат: Статистика по объявлению с указанным id

5. Удалить объявления с указанным id
Желаемый результат: 200	ok

6. Проверка соответствия параметров создаваемого объявления и созданного
Желаемый результат: Создаваемый на клиенте и создаваемый на сервере объявления должны совпадать

7. Создание пустого объявления
Желаемый результат: 400 bad request

8. Удаление уже удалённого объявления
Желаемый результат: 404 not found

9. Проверка типов данных
Желаемый результат: Невозможность отправки запроса с неподходящими типами данных

10. Проверка на ввод отрицательной цены
Желаемый результат: Нельзя ввести отрицательную цену
