Для запуска тестов необходимо иметь установленный Python. (Эти тесты создавались на Python 3.13.3 на ОС Windows 10)

1. Скачайте любым удобным образом в нужную вам директорию файлы репозитория или скопируйте его командой:

```
https://github.com/MousePos/avito-qa-internship.git
```

2. В нужной вам директории создайте виртуальное окружение:

```
python3 -m venv venv
```

3. Активируйте виртуальное окружение командой:
```
./venv/Scripts/Activate
```

4. Установите зависимости

```
pip install -r requirements.txt
```

5. Запустите тесты

```
pytest test_api.py -v
```
